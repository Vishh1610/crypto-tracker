#Installation and Local Run

The application runs purely on the client-side. No complex backend setup is required.

1.  Clone the Repository:
    ```bash
    git clone [https://github.com/Vishh1610/crypto-tracker.git](https://github.com/Vishh1610/crypto-tracker.git)
    cd crypto-tracker
    ```

2.  Run:
    Open the `index.html` file directly in your preferred web browser.

    (Use the VS Code "Live Server" extension for the best experience.)
